# 2023 빅콘테스트 정형데이터 어드밴스드 리그 (우수상)

## 개요
- 팀명: 전당포
- 프로젝트 기간: 2023.08 - 2023.11
- 팀원: 3명
- 주제: 클래식 공연 활성화를 위한 예술의전당 콘서트홀의 효과적 가격 모델 수립

	3. 좌석별 후기 텍스트를 활용한 군집화
